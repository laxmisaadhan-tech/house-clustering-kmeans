# House Clustering using K-Means

## Objective
In this project, I applied K-Means clustering to group houses based on different features such as income, population, and housing characteristics. The goal is to identify patterns in housing data.

## Dataset
California Housing Dataset (from Scikit-learn library)

## Approach
K-Means clustering algorithm was used to group similar houses after scaling the dataset. The optimal number of clusters was determined using the Elbow method, and PCA was used for visualization.

## Steps Involved
- Data preprocessing
- Feature scaling using StandardScaler
- Finding optimal number of clusters using Elbow Method
- Applying K-Means clustering algorithm
- Visualizing clusters using PCA

## Results
The dataset was successfully divided into 3 clusters. Each cluster represents different types of housing groups with varying characteristics.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
